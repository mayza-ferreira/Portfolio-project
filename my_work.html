FSW Barber

The FSW Barber App was a challenging project I built during Full Stack Week. I worked with Next.js, React.js, Node.js, TypeScript, PostgreSQL, and Tailwind CSS. One of the biggest challenges was integrating the front-end with the back-end, which helped me better understand the architecture of a full-stack application. I also learned how to handle real-time data and use TypeScript to write more robust code. The app's design was provided by the course, based on a Figma prototype, which I used as a reference to build a visually appealing and user-friendly interface. The design follows a mobile-first approach, ensuring a seamless experience across all devices. The app is deployed on Vercel for easy access and performance optimization. Source code
